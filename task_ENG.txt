Task: create a firmware for Arduino Nano 

Context:
Arduino game. From one end of the strip, the user generates the corresponding color using buttons to match the one falling from the other end of the strip. LED strip 30 - 60 pixels (diodes) Top of the strip - the opposite end from the user Bottom - Arduino + the beginning of the strip 

Multi-colored pixels fall from top to bottom one by one The color is one of three [red, blue, green] Empty distance between lit pixels is random [1 - 5] Falling speed by levels: 

    the level is changed by a button. 

    the first level - 2 pixels per second 

    total of 5 levels. 3 pixels per second, 4, etc. After 50 accurate shots - the level automatically increases. 

Buttons: (joystick later) 

    Red generates a red pixel 

    Blue generates a blue pixel 

    Green generates a green pixel 

Upon pressing - the generated pixel rises at the same speed as the pixel falling from above. 

Logic: 1 If the color of the pixel generated by the user matches the first pixel on the path - the light disappears at the moment of collision. 

2 If the color of the pixel generated by the user does NOT match the first pixel on the path - this pixel changes its color to a random one from the list and continues its path downwards. 

3 If the first falling pixel reaches the very bottom -  the level resets to the first generation of pixels starts over from the very top 

Hardware: Arduino Nano 4 buttons or a joystick speaker (later) LED strip 5050 or similar battery 

What pull-up resistor is needed for the buttons? Buttons are connected: GREEN_BTN D2 RED_BTN D3 BKUE_BTN D4 LEVEL_UP_BTN D5 

What resistor is needed for the strip? Strip is connected: LED_STREEP D6
